What is this?

This is a browser-based editing system for audio and video. It allows you to edit a recording using a transcript of what was said in the recording. The transcript is automatically generated when you upload the file.

Who made it?

This system was created by Chris Baume in BBC R&D's audio team. It relies on speech-to-text technology developed by IRFS in work sponsored by Rewind (BBC Northern Ireland).

Can I use it?

Anybody with a BBC login on the NATIONAL domain can log in and use it.

Is it secure?

Yes, all of your data is transmitted and stored on internal BBC networks and systems.

Is it reliable?

No, this is an experimental platform which is not yet properly supported.

How does it work?

You can find out by looking through the code which is on Github.
